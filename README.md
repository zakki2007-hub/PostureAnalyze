# 作品介绍：智能坐姿检测软件---坐姿管家
郜金智、赵星渊


**一、介绍与灵感**

  作品灵感来源于生活：在长时间的学习或者工作后，人们的坐姿通常不是很正确，而且很容易久坐，这时不仅工作或者学习结束后人们通常会感到全身酸痛，还对脊柱、颈椎都有一定的损伤，基于此，本小组研发了一款可以提醒人们久坐或坐姿不良的软件，通过树莓派+MediaPipe库分析坐姿，并将数据传输到手机app上，让人们能实时看到自己是否坐姿良好，并且在久坐或者坐姿不良的时候进行提醒


**二、原理介绍**

  “坐姿管家”通过树莓派+摄像头获取图像数据，通过OpenCV库处理图像，再用mediapipe库画出人体骨骼，取右耳、右肩、髋部三个部位。分别计算耳与肩、肩与髋之间的角度，当角度超过一定阈值就判定为探头或者驼背，进而提醒用户调整坐姿。数据通过局域网传输给手机APP，与之相配的手机APP通过Flutter进行开发与封装，并且当人体出现在画面中超过60分钟时判定为久坐，提醒用户起来活动一下


**三、现场反响**
  
  坐姿管家在现场吸引了不少人在现场进行体验，反响极为热烈，我们也发现了该产品对不同人群的体验效果并不相同：身高比较高的人，检测精度更差；而身高较低的人，产品检测不良坐姿时更灵敏。此外，如果有头发遮挡耳朵或者面部的情况，本产品甚至无法识别到摄像头里出现了一个人。针对这些不足，现场的观众也提出了很多建设性的建议：不少老师身为家长，对这部产品表现出更加浓厚的兴趣，希望将其运用到纠正孩子写作业时候的场景当中；也有人提出可以将摄像头放置在正面，通过调整算法来完成坐姿检测，这样不仅更方便，还可以直接运用笔记本上的摄像头。总而言之，现场的同学与老师普遍认为这一产品不仅是一个技术作品，更是体现人文关怀的健康解决方案，有着广泛的应用场景与巨大的潜力。
